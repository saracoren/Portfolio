Quality Assurance and Project Management

I worked extensively to coordinate between project managers and developers and make sure we completed the project to the highest standards. Being that this was a multi site build with an ecommerce element, there were a lot of moving parts to keep track of throughout the project. I kept everyone on task and organized by logging all outstanding items in Notion. I created tags for differnt line items indicating their status and correlating site. I also included screenshots to clearly communicate outstanding issues as well as links to staging urls and a column for comments so we could all communicate regarding the status of items.





Extensive Testing

I was additionally responsible for creating a comprehensive test plan to ensure that all ecommerce and user authentication features were working as desired. I also tested all forms and surveys and managed contacts in Keap/ Infusionsoft and routinely checked all pages and features accross browsers and devices using BrowserStack.
