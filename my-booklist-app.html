My Booklist

Live Site

GitHub Repo

My Booklist is a simple app that can be used by anyone who wants to keep track of the books they want to read and the books they have finished reading. However, as a former elementary teacher, my original inspiration was for emerging readers to use this app as a tool to be prepared for idependent reading time and to make sure they persevere to finish one book before moving on to the next one.


Technologies Used

HTML5

CSS3

JavaScript

jQuery

Google Books API

Process

I built this app for the first project I completed in my Software Engineering Immersive course. I wanted to built something that fulfilled a need, and as a former elementary teacher, I thought using an app like this could be very useful in helping students prepare for independent reading time and encourraging them to finish books instead of abandoning them midway through. I drew up a quick sketch and then began coding my app so I would have plenty of time left over towards the end to trouble shoot any issues and have ample time to polish my CSS.



Shortly after completing this project, I visited my former students at school and shared this app with them. They were very excited about it and I was pleased to see that my goal of creating a useful classroom app had been accomplished.
