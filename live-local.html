Live Local

Live Site

GitHub Repo

Live Local is an app that utilizes the Google Maps API. Users can enter in local events to notify members of their community about parades, picnics, fireworks, festivals and any other kind of event imaginable. Events can be viewed in map view or in list view. The map view is particulaly useful for users who want to see what is happening in any given geographic area on the day of interest.
